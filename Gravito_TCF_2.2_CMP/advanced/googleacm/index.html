Google Additional Consent Mode

Introduction 

This documentation provides a step-by-step guide on enabling Additional Consent Mode (ACM) support in Gravito TCF CMP. ACM is a feature that allows your CMP to handle additional consents required by Google.  You can find the Google technical specification about ACM here.

What is Additional Consent Mode (ACM)?

Additional Consent Mode (ACM) is a feature introduced by Google to manage additional user consents for third-party vendors, particularly for scenarios like ad personalization. 

Enabling ACM Support

Login to Admin Portal 

  1. Open your web browser and navigate to the Gravito TCF CMP Admin Portal. 

  2. Log in with your credentials. 

Configure a New CMP 

Basic Tab

  1. In the Admin Portal, select the "CMP" option from the navbar. 

  2. Choose "TCF 2.2 Configurator" from the available options. 

  3. For a new configuration, click on the "New Configuration" button. 

  4. Fill in the required details on the Basic tab and click "Next" to move to the Settings tab. 

Settings Tab

  1. On the Settings tab, fill in the required fields as per your configuration requirements. 

  2. Click on the "Use ACM Mode" checkbox to enable ACM support. 

  3. Click "Next" to proceed to the Core tab. 

Core Tab

  1. In the Core tab, find the "Google Vendors" multi-select field. 

  2. Select the Google vendors you want to use. 

  3. Configure other fields based on your requirements. 

  4. Click on the "Preview" button. 

Preview CMP

  1. In the preview's second layer, navigate to the "Vendors" tab. 

  2. You will see the selected Google vendors listed under vendors. 

Add ACM Support to Existing Config 

  1. Follow the steps to reach the configuration list in the Admin Portal. 

  2. Click on the "Copy Config" button for the existing configuration you want to update. 

  3. Follow the steps outlined above for configuring a new CMP, starting from the Basic tab. 

This completes the process of enabling Additional Consent Mode (ACM) support in Gravito TCF CMP. If you encounter any issues or have questions, please reach out to our support team.
